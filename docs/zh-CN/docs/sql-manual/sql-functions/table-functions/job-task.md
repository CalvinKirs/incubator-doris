## `tasks`

### Name

<version since="dev">

tasks

</version>

### description

表函数，生成 tasks 临时表，可以查看当前 doris 集群中的 job 产生的tasks 信息。

该函数用于from子句中。

#### syntax

`tasks("type"="insert");`
**参数说明**

| 参数名  | 说明     | 类型     | 是否必填 |
|:-----|:-------|:-------|:-----|
| type | 作业类型   | string | 是    |

***type 当前仅支持 insert 或者 mv***
